FAQ - unkayOS

O que é o unkayOS?

É um sistema operacional web, modular, que simula um desktop no navegador, permitindo múltiplos apps, widgets e personalização.

Como crio um novo app?

Siga o guia de apps e use a estrutura recomendada em apps/.

Posso rodar o unkayOS offline?

Sim, basta abrir o index.html em um navegador moderno.

Como adiciono meu app ao menu iniciar?

No config.json do app, defina hidden: false e adicione ao apps/apps.json.

Como funciona o isolamento de estilos?

O core aplica um namespace CSS automático para cada app, evitando conflitos.

O que acontece se um app travar?

O AppManager permite encerrar e reiniciar apps individualmente, sem afetar o sistema.

Como contribuo com o projeto?

Veja o CONTRIBUTING.html.

Onde reporto bugs ou peço features?

Abra uma issue no repositório ou envie um PR.

Quais navegadores são suportados?

Chrome, Firefox, Edge, Safari (desktop). Mobile não é oficialmente suportado.

Como removo um app?

Remova a pasta do app em apps/ e sua entrada em apps/apps.json.

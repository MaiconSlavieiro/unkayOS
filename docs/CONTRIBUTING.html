Guia de Contribuição - unkayOS

Obrigado por querer contribuir com o unkayOS! Siga estas orientações para facilitar a colaboração:

Como contribuir

  1. Faça um fork do repositório
  2. Crie uma branch descritiva (ex: feature/novo-app ou fix/bug-taskbar)
  3. Faça commits claros e objetivos
  4. Abra um Pull Request detalhando sua contribuição

Padrão de Commits

  • Use prefixos: feat:, fix:, docs:, refactor:, test:, chore:
  • Exemplo: feat: adicionar widget de clima

Sugestão de Apps

  • Crie uma pasta em apps/ seguindo o guia de apps
  • Adicione ao apps/apps.json
  • Inclua exemplos e documentação mínima

Reportando Bugs

  • Abra uma issue detalhando:
    • Passos para reproduzir
    • Comportamento esperado
    • Prints ou logs, se possível

Boas Práticas

  • Siga o padrão de código do core
  • Use variáveis do design-system
  • Teste seu app em diferentes resoluções
  • Sempre limpe timers e listeners no onCleanup

Dúvidas?

Consulte o FAQ.html ou abra uma issue!
